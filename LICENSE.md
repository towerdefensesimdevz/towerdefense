**❌ Problem: GUI doesn't appear after execution**

**❌ Problem: Auto farm gem loop stops mid-session**
✅ **Fix:** Increase `FARM_LOOP_DELAY` in your config from `1500` to `2500`. Some servers throttle action speed detection at lower intervals.

**❌ Problem: Unlock all feature not working**
✅ **Fix:** Ensure `UNLOCK_ALL_ON_START` is set to `true` in settings, or manually trigger it via the GUI's Unlock tab. Some content requires the game to be in a lobby state.

**❌ Problem: Narrator not speaking**
